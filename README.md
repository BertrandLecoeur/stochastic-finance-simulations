# Stochastic Finance Simulations

Quantitative finance project implemented in R.

## Project Overview

This project focuses on stochastic processes and continuous-time finance methods used in quantitative finance.

The project includes:
- Brownian motion simulation,
- Monte Carlo methods,
- stochastic differential equations (SDEs),
- Black-Scholes option pricing,
- finite difference methods for derivatives pricing.

---

## Topics Covered

### Brownian Motion Simulation
- Simulation of standard Brownian motion paths
- Correlation estimation
- Distribution analysis
- Monte Carlo variance estimation

### Stochastic Differential Equations
Simulation of SDEs using:
- Euler-Maruyama discretization
- Improved exponential discretization schemes

### Option Pricing
Implementation of:
- Black-Scholes pricing formulas
- European call and put pricing
- Strap option pricing
- Finite difference pricing methods
